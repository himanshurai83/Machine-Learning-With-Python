# 📊 EDA Dashboard with Streamlit

An interactive **Exploratory Data Analysis (EDA) Dashboard** built using **Streamlit, Pandas, Seaborn, and Plotly**.
Upload any CSV file and instantly explore your dataset with statistics, visualizations, and automated insights.

---

## 🚀 Features

### 📋 Overview

- Dataset shape (rows × columns)
- Numerical & categorical column detection
- Missing values & duplicates analysis
- Column-wise detailed summary
- Dataset preview
- Download processed dataset

---

### 📈 Distributions

#### Numerical Analysis

- Histogram (interactive)
- Box plot (outlier detection)
- Scatter plot (relationship between two variables)

#### Categorical Analysis

- Bar chart (top categories)
- Pie chart distribution
- Aggregation analysis (mean, median, sum)

---

### 🔗 Correlation Analysis

- Full statistical summary (`describe()`)
- Correlation heatmap for numerical features

---

### 🔍 Automated Insights

- Strong correlations detection
- Top-performing categories
- Most frequent category combinations
- Best & worst performing groups
- Outlier detection using IQR

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```

## 📥 Usage

1. Launch the app
